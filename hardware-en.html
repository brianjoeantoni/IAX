Hardware

Hardware

The IAX Series carries the IAX-1502 as a network-based controller can control one to two readers. IAX-1502 has Small dimensions so it is easy to install. Not only compact size, IAX-1502 has high performance and Capable of storing up to 1,000,000 credentials in standalone mode or not connected to the network.

IAX-1502 also has high flexibility and can supports readers with RFID technology, magnetic stripe, biometrics (fingerprint, face and iris), PIN code, QR code and also reader with Bluetooth and NFC technology. This IAX-1502 controller too equipped with IA-BUS interface for additional input contacts and output relays if needed.

IAX-1000

Network Controller for Contacts & Relays

The IAX-1000 is a master controller having an IA-BUS interface and can be used to control 8 IAX-1208 controllers or 8 IAX-1308 controllers. Number of IAX-1000s that can be controlled by one software application is unlimited. Matter this makes the IAX-1000 system usable from a small system for example to monitor several input contacts to a system large for example to monitor thousands of input contacts at once controlling thousands of dry contact relays.

IAX-1208

Child Controller For Login Contacts

IAX-1208 is a child controller that has 8 contact inputs dry. Dry contacts are generally used as the output of sensors. For example, a sensor to detect the condition of a the door is being closed or opened. This controller has a IA-BUS interface that can be connected to the master controller such as IAX-1000 or IAX-1502. An IA-BUS interface can be supports up to 8 child controllers.

IAX-1308

Child Controller For Output Relay

IAX-1308 is a child controller that has 8 contact relays dry. Dry contact relays are generally used as switches to activate or deactivate electronic devices by means connect or disconnect the power supply. Dry contact relay for example, it can be used to flow or break the flow electricity to the electronic lock so that we can lock or Open a door using an electronic key.

This controller has an IA-BUS interface that can connected to a master controller such as the IAX-1000 or IAX-1502. An IA-BUS interface can support up to 8 units child controller.

IAX-1502

Network Controller for Two Reader Access

The IAX-1502 is a network-based controller that can controlling two reader accesses. This controller has dimensions which is small and compact so it can be installed easily. The IAX-1502 is suitable for controlling two doors with single reader configuration or can be used for controlling one door with dual reader configuration.

This controller offers great flexibility and can supports readers with RFID technology, magnetic stripe, biometrics (fingerprint, face and iris), PIN code, QR code and also reader with Bluetooth and NFC technology.
